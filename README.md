Student Marks Calculator

Problem Statement...
Many students and beginners face difficulty in manually calculating:
Total marks
Percentage
Grade
Pass/Fail status
Doing these calculations repeatedly is time-consuming and may lead to mistakes, especially when multiple subjects are involved.

Solution
The Student Marks Calculator is a simple web-based mini project built using HTML, CSS, and JavaScript that:
Takes student name and subject marks as input
Automatically calculates:
Total marks
Percentage
Grade
Pass/Fail status
Displays results dynamically on the browser
Provides a clean, centered UI with a professional navbar
Includes a reset option for new entries
This project helps beginners understand DOM manipulation, basic logic, and dynamic UI updates using JavaScript.

 Features
Student name input
5 subject marks input
Automatic total calculation
Percentage calculation
Grade system (A+, A, B, C, D, F)
Pass / Fail result
Fully dynamic output (no console)
Clean and responsive design
